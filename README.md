VisionX

Student Budgeting & Expense Tracking App

A lightweight, user-friendly mobile budgeting application 
designed specifically for university students to track expenses, manage budgets, and build better financial habits.


Problem Statement

University students often live on tight budgets — scholarships, part-time jobs, or family support — yet most have no system to track where their money goes. 
Without visibility into daily spending, it is easy to overspend, miss savings goals, and build poor financial habits. 
Existing budgeting apps are either too complex or designed for working adults, not students.
VisionX fills that gap with a clean, no-login, instant-use budgeting tool built specifically for students.


Features                       Description

Dashboard              Live net balance, income vs expense summary, savings rate
Add Transactions       Log income and expenses with categories and dates
Budget Limits          Set monthly spending caps per category with visual progress bars
Analytics              Bar charts, doughnut breakdown, 6-month income vs expense trends
Search & Filter        Instantly search or filter transactions by type
Data Persistence       All data saved locally — survives page refreshes
Export Data            Download your transactions as CSV or full JSON backup
Profile Settings       Customize your name and preferred currency symbol
Mobile-First           Responsive design optimized for phone screens

Usage
Adding a transaction

Tap the green + button at the bottom right
Select Income or Expense
Fill in the description, amount, category, and date
Tap Save Transaction

Setting a budget limit

Go to the Budget tab
Tap Add Budget Limit
Choose a category and enter your monthly limit
Your spending progress will be tracked automatically

Viewing analytics

Go to the Analytics tab
View spending by category (bar chart), expense split (doughnut), and 6-month trend (line chart)

Exporting data

Go to Settings
Tap Export as CSV to open in Excel or Google Sheets
Tap Export as JSON for a complete data backup
